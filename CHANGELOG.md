## [1.0.0] - 2024-11-25

### Added

- Initial release of React Native Brownfield Integration
- MyRNFramework: React Native framework source with demo app
  - Counter demo component
  - Input demo component
  - Features list display
  - TypeScript support
- iOS native host app demo (`ios/`)
  - SwiftUI and UIKit integration examples
  - Push navigation, modal presentation, and embedded view demos
  - Uses `@callstack/react-native-brownfield` library
- Android native host app demo (`android/`)
  - Jetpack Compose integration
  - Full-screen and embedded React Native view demos
  - Uses `@callstack/react-native-brownfield` library
- Comprehensive README with:
  - Architecture overview diagram
  - Project structure documentation
  - Quick start guide
  - AAR/XCFramework usage instructions
  - Integration examples for iOS and Android

### Dependencies

- React Native 0.73.0
- React 18.2.0
- @callstack/react-native-brownfield (latest)
- Hermes JavaScript engine
